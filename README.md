# Double-Incomplete Multi-view clustering with Self-induced Semantic Label Diffusion

This repo contains the code of the paper: DISSLD

## Requirements

tensorflow==2.10.0

numpy>=1.26.4

scikit-learn==1.6.1

python-box==7.3.2

## Training

The dataset and hyperparameters can be changed in `main.py`

The dataset settings are in `dataset_config.py`

Run the code by

```bash
python main.py
```
